ViewPort 단위

Vh
Vw
%

vw (Viewport Width)

  • "vw"는 뷰포트의 너비를 기준으로 합니다.
  • 뷰포트의 너비를 100등분한 단위!
  • ex) 1vw = 1% 너비
  • 화면의 너비에 따라 크기가 조정!

vh (Viewort Height)

  • "vh"는 뷰포트의 높이를 기준으로 합니다.
  • 뷰포트의 높이를 100등분한 단위!
  • ex) 1vh = 1% 높이
  • 화면의 높이에 따라 크기가 조정!

% (Percentage)

  • "%"는 부모 요소의 크기를 기준으로 합니다.
  • 부모 요소의 크기를 100 등분한 단위!
  • 부모 요소의 크기에 따라 크기가 조정!

calc()함수

  • calc()함수는 다양한 단위를 사용하여 길이, 너비, 높이 등을 계산하는데 사용!
