index.js가 index.html에 보여줌.


웹 애플리케이션의 다른 페이지들은 React.js를 사용하여 만들어집니다.

이 페이지들은 정적으로 HTML 파일에 작성된 것이 아니라, JavaScript를 통해 동적으로 생성됩니다.

애플리케이션의 초기 HTML 파일(index.html)에는 <div id="root"></div>라는 빈 요소만 포함되어 있습니다.
애플리케이션이 실행되면, React는 이 root 요소를 찾고, ReactDOM을 사용하여 해당 요소 안에 필요한 모든 컴포넌트를 렌더링합니다.
이 div 요소는 React 컴포넌트가 렌더링 되는 "루트"역할을 함.

결과적으로 사용자가 보는 모든 페이지와 UI 요소들은 이 root 요소 안에 동적으로 삽입되고 업데이트됩니다.
